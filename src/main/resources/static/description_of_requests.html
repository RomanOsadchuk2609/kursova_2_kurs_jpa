Опис запитів
№ Опис
1.1 Одержати перелік спортивних споруд вказаного типу
1.2 Одержати перелік спортивних споруд вказаного типу, в яких можливе проведення змагань з певного виду спорту
2.1 Одержати список спортсменів, які займаються вказаним видом спорту
2.2 Одержати список спортсменів, які займаються вказаним видом спорту, не нижче певного розряду
3.1 Одержати список спортсменів, які тренуються у певного тренера
3.2 Одержати список спортсменів, які тренуються у певного тренера, не нижче певного розряду
3.3 Одержати список спортсменів, які тренуються у певному спортивному клубі
4 Одержати список спортсменів, які займаються кількома видами спорту, вказавши всі види
5 Одержати список тренерів вказаного спортсмена
6.1 Одержати список змагань проведих в певний період
6.2 Одержати список змагань проведих в певний період та вказаним організатором
7.1 Одержати список призерів вказаного змагання
7.2 Одержати список учасників вказаного змагання та місця, які вони зайняли
8.1 Одержати список змагань проведих у вказаній спортивній споруді
8.2 Одержати список змагань проведих з певного виду спорту
9 Одержати перелік спортивних клубів та кількість спортсменів цих клубів, які брали участь у спортивних змаганнях протягом вказаного періоду
10 Одержати список тренерів з певного виду спорту
11 Одержати список спортсменів, які протягом вказаного періоду не брали участь у жодному змаганні
12 Одержати список організаторів змагань та кількість змагань, які були ними проведені протягом вказаного періоду
13 Одержати перелік спортивних споруд та дати проведення на них змагань протягом певного періоду
